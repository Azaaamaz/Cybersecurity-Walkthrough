# JavaScript, what and how?

## 1. Overview

JavaScript allows page to become interactive. HTML used to create website structure, while JavaScript used to control the functionality of web pages. JS can dynamically update the page in real-time, giving functionality to change the style of a button when a particular event on the page occurs (such as when a user clicks a button) or to display moving animations.
